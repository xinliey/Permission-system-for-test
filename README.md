# Permission-system-test
Permission assurance system created by C# dotnet and Angular 

# Users can
- view project requests
- pprove / reject requests
- add approval remarks
- track project status

# Tech Stack
Front End : Angular , TypeScript , HTML/CSS
Back End  : ASP.NET core web api , C#, MySQL

# Feature 
- display project lists from database
- select project approval and rejection
- Add remark via modal pop up
- update project status to : อนุมัติ ไม่อนุมัติ
- disable check box after update status
- sync data with backend API
